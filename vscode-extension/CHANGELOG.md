# Changelog

All notable changes to Timeline Workflow Preview are documented in this file.

## 0.2.0

- Follow the VS Code display language in English and Japanese.
- Localize empty states, workflow errors, default titles, and time labels.

## 0.1.0

- Preview the first `workflow` block in the active Markdown document.
- Update the preview automatically while editing or switching Markdown documents.
- Display missing-block guidance and workflow syntax errors in the preview.
- Reuse the parser, layout engine, and SVG renderer from the web application.
